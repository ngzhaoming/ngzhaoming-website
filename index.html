Projects Done

SPLIT-IT


During my first summer break, I took part in a school summer project, Orbital. The project involved teaming up with a partner and developing a software product. Our pair developed a bill-splitting mobile application, Split-It. The mobile application is developed using React-Native. Hence, it is compatible in both Android and iOS.

Split-It aims to convenience the bill-splitting process. The main features of it includes the Optical Character Recognition (OCR) feature which allows users to scan their receipts. While developing the application, we ensure that it will cater more towards the Singapore context. This includes different payment options to settle the bills, such as through cash or E-wallet.
